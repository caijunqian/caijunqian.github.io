1.知识储备

快速幂原理

当b为偶数时，a ^ b = (a ^ (b/2) ) * (a ^ (b/2) )

当b为奇数时，a ^ b = (a ^ (b/2) ) * (a ^ (b/2) ) * a

这个不难理解，举个例子

3 ^ 6 = (3 ^ 3 ) * (3 ^ 3) 此时b=6为偶数

3 ^ 5 = (3 ^ 2 ) * (3 ^ 2) * 3 此时b=5为奇数

根据这两个等式，当我们在算a ^ b时可以先算出 a ^ (b / 2)，然后再求a ^ b，减少了重复的b / 2次计算，因此时间复杂度是O(log N)

a # b % c = ( a % c # b %c ) % c 其中 # 代表运算加、减、乘。
