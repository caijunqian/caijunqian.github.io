bfs算法思想

通常利用队列实现：

Q={初始状态start};
标记start为已访问;
while(Q非空)
{
队首元素t出队列;
if(t为目标状态){算法结束;}
所有与 t 相邻 的 未访问 的状态next入队列
标记next为已访问;
}
